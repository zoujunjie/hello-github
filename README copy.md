# Hello GitHub

A simple React demo to search and display GitHub repositories, built with React, Vite, TypeScript, and Ant Design.

## Prerequisites

- Node.js
- npm

## Initialization

Install the required dependencies:

```bash
npm install
```

## Running the Application

To start the local development server:

```bash
npm run dev
```

This will launch the application (by default at `http://localhost:5173/`).

## Building and Publishing

To build the application for production:

```bash
npm run build
```

This command will compile the code and bundle the application into the `dist` directory. 

To preview the built production application locally:

```bash
npm run preview
```
